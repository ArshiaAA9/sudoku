Sudoku game made for the terminal with the ftxui library in c++.

# Installation
Cloning and building the repo:
```
git clone https://github.com/ArshiaAA9/sudotu
cd sudotu
cmake -B build
```
Running the game:
```
cd build
./sudotu
```

# Controls
Use 1-9 numbers to make a move in empty cells.
Move with WASD or arrow keys to move.
